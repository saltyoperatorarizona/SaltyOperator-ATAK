# ATAK Plugins by Stephan Pellegrini

Free, open-source plugins for ATAK (Android Team Awareness Kit) built for the operator community.

---

## Plugins

| Plugin | Version | Description | Status |
|--------|---------|-------------|--------|
| [OSINT](./Plugins/OSINT/) | 5.6 ATAK-CIV | Open source intelligence feed aggregator. Consolidate news, threat intel, geopolitical analysis and regional reporting directly in ATAK. Supports 30 custom RSS feeds or 80 curated presets across Defense, Intel, Geopolitics and Regional categories. Works worldwide. | Available |
| [AIS-D](./Plugins/AIS-Direct/) | 3.0 ATAK-CIV | Real-time AIS vessel tracking plugin. Monitor maritime vessel traffic worldwide directly in ATAK. Displays vessel positions, headings, speeds, and call signs on the ATAK map. Supports AISStream, MarineTraffic, and NMEA/TCP data sources with military vessel quick filter. | Available |
| [CODEC](./Plugins/CODEC/) | 5.6 ATAK-CIV | Team biometric uplink plugin. Monitors real-time heart rate data from BLE heart rate sensors and displays a military-style HUD for the entire team. Supports Polar, Wahoo, Garmin, Coospo, and any standard BLE HR monitor. MGSV-inspired CODEC interface with EKG waveforms, KIA detection, and CoT broadcast. | Awaiting Tak.Gov Approval |
| [CamFeed](./Plugins/CamFeed/) | 1.0 ATAK-CIV | Bulk public camera feed aggregator. Queries Windy Webcams API, OpenStreetMap, and local KMZ/KML files to discover public cameras near any map location. Results display as CoT markers on the ATAK map with built-in stream viewer. | Awaiting Tak.Gov Approval |
| [OptimalZero](./Plugins/OptimalZero/) | v19 ATAK-CIV | G1 ballistics engine that calculates the optimal zero distance for Maximum Point Blank Range (MPBR). Supports 5.56, 7.62x51, 7.62x39, 300 BLK (supersonic and subsonic), .277 Fury/6.8x51, 6.5 CM, 5.7x28, 9mm, and 45 ACP. Returns optimal zero, MPBR, drop table, and full trajectory chart. Barrel lengths 4"â24" covering PDW to precision rifle. | Available |
All plugins are free, open-source, and built for the operator community. Additional plugins may be added over time.
